================================================================================
 SECTION 2: THE PHILOSOPHY — RULE-BASED FIRST, AI ONLY WHERE REQUIRED
================================================================================

THE GOLDEN RULE:
  Use RULES for repetitive, predictable tasks.
  Use AI ONLY for tasks that need judgment, personalization, or creativity.

WHAT IS RULE-BASED (NO AI NEEDED):
═══════════════════════════════════════════════════════════════════════════════

  ✓ Follow-up scheduling (if deal stage = "proposal" → remind in 3 days)
  ✓ Email sending (trigger template at scheduled time)
  ✓ Task creation (when lead created → assign to salesperson)
  ✓ Status updates (if email opened → change activity)
  ✓ Tag assignment (if source = "website" → tag = "inbound")
  ✓ Pipeline movement (if deal value > ₹1L → move to "priority" stage)
  ✓ Notifications (if deal inactive 5 days → notify owner)
  ✓ Data import/export (CSV parsing, field mapping)
  ✓ Duplicate detection (same email + same name = possible duplicate)
  ✓ Backup scheduling (every 30 days → trigger backup)
  ✓ Trial expiry (if trial_ends_at < NOW() → downgrade)
  ✓ Rate limiting (max 100 requests/hour → reject)
  ✓ Usage tracking (count API calls → update meter)
  ✓ Email sequence execution (step 1 sent → wait 3 days → send step 2)

WHAT NEEDS AI (JUDGMENT REQUIRED):
═══════════════════════════════════════════════════════════════════════════════

  ✓ Lead scoring (analyzing patterns, weighing factors — needs judgment)
  ✓ WhatsApp replies (understanding intent, handling objections — needs NLP)
  ✓ Follow-up EMAIL CONTENT (personalized message — needs creativity)
  ✓ Proposal content generation (scope, executive summary — needs writing)
  ✓ Content generation (posts, blogs, ad copy — needs creativity)
  ✓ Voice call conversations (real-time understanding — needs speech AI)
  ✓ Sentiment analysis (reading tone from text — needs NLP)
  ✓ Revenue gap detection (pattern recognition across data — needs analysis)
